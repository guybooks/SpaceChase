Spacechase

An Open Source, Cross Platform, Real-time Multiplayer Game

Download .zip Download .tar.gz View on GitHub

Space Chase is an open source, cross platform, mobile multiplayer sample game created by Nextpeer. The game demonstrates Nextpeer’s SDK main features. The tutorials series that comes together with the game’s code explains how to implement the different features in code.

Sign up:

There will be a sign up form that allows users to enter their email which will be submitted to us in order to receive the source code. Text: “Sign up to get your free source code”

Features list:

  • Real time multiplayer
  • Cross platform iOS & Android
  • Live view of opponents on screen
  • Power Ups
  • Ghost players based on past real player recordings
  • Friends challenges via Facebook
  • Multiple levels which are all in sync across devices
  • Social identity
  • Multiplayer leaderboards
